# ArXiv Scientific Trends Analysis

Unsupervised learning project analyzing 2.4M ArXiv papers to discover research trends.

## Project Structure

- `01-04_preprocess`: Data loading and cleaning
- `05-06_features`: TF-IDF and dimensionality reduction
- `07-08_model`: Clustering with K-Means
- `09-11_analysis`: Results and visualizations
